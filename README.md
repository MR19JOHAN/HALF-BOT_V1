## neoxr-bot
WhatsApp Bot (Base)

### Database Setup

Create a heroku account, then go to the AddOns page and add Heroku Postgres.

### Installation

```
$ git clone https://github.com/neoxr/neoxr-bot
```

And then go into the folder and type this on your console.
```
$ npm i
$ node . <session_name>
```
